Chrome Extension for ChatGPT + Google Drive

ChatGPT Auto Save to Google Drive

Automatically detect changes in your ChatGPT conversations and save the latest snapshot to your own Google Drive as a Markdown file. No developer-controlled server is used.

View Privacy Policy View GitHub Repository
  • Reads visible ChatGPT conversation content directly from the page DOM.
  • Names files as ChatGPT title plus date, creates a new file for a new title, and merges same-title content into the same daily Markdown file.
  • Uses Google OAuth and the safer drive.file scope.
  • Stores data only in the user's own Google Drive account.

Example Extension Status

Authorization Authorized
Sync Status Watching for changes
Target Folder ChatGPT-AutoSave
Output Format Markdown

Single Purpose

The extension exists for one purpose only: saving ChatGPT conversation content to the user's own Google Drive as Markdown files.

Permissions

Uses identity, storage, tabs, and required host permissions for ChatGPT pages and Google Drive API requests.
