Review steps for Chrome Web Store:

Chrome Web Store item ID: dmfmnjinpphfpodandcboelcglicaomg

1. Install the extension.
2. Click the extension icon to open the popup.
3. Click "Connect Google Drive".
4. Approve the Google OAuth prompt for the signed-in Chrome profile.
5. After approval, the popup shows:
   - Google Drive: connected
   - A link named "Open Google Drive folder"
6. Click the folder link to confirm that the extension created or reused the "AI Chats Auto Save" folder in the reviewer's own Google Drive.
7. Open any supported AI chat page, for example https://chatgpt.com/.
8. Send or edit a message in a conversation and wait a few seconds.
9. A Markdown file is created or updated under "AI Chats Auto Save/<Platform>" in the reviewer's own Google Drive.

Notes:
- The extension does not use any developer-controlled server.
- All uploads go directly from the browser to the reviewer's own Google Drive account.
- The extension uses the limited Google Drive scope: https://www.googleapis.com/auth/drive.file
- If Drive access has not been granted yet, syncing stays paused until the reviewer clicks "Connect Google Drive" in the popup.
